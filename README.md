# 💻 C Programming Fundamentals — Complete Programs Collection

This repository contains a comprehensive collection of C language programs developed while studying Programming Fundamentals. It covers all core concepts of C programming, ranging from basic syntax to problem-solving and logic building.


## 📚 About This Repository

This repository is designed to help beginners understand the fundamentals of C programming through practical examples. Each program demonstrates a specific concept and helps in building a strong programming foundation.

## 🚀 Topics Covered

### 🔹 Basic Concepts

* Input / Output (printf, scanf)
* Variables and Data Types
* Operators (Arithmetic, Relational, Logical)
* Type Casting

### 🔹 Control Structures

* If / Else Statements
* Nested If
* Switch Case
* Conditional (Ternary) Operator

### 🔹 Loops

* For Loop
* While Loop
* Do While Loop
* Nested Loops

### 🔹 Functions

* User-defined functions
* Function declaration & definition
* Parameter passing
* Return values
* Recursion (basic)

### 🔹 Arrays

* 1D Arrays
* 2D Arrays (Matrices)
* Array operations (sum, max, min, reverse)

### 🔹 Pointers (Basics)

* Pointer declaration
* Pointer arithmetic
* Call by reference

### 🔹 Structures

* Structure definition
* Accessing members
* Array of structures

### 🔹 File Handling

* File opening and closing (fopen, fclose)
* Reading from files (fscanf, fgets, fgetc)
* Writing to files (fprintf, fputs, fputc)
* File modes (r, w, a, r+, w+, a+)
* Working with text files
* Basic file-based programs

## 🧪 Sample Programs

* Prime number
* Factorial calculation
* Fibonacci series
* Palindrome number & string
* Matrix addition & multiplication
* Searching & sorting (basic)
* File read and write operations

## ⚙️ How to Run

1. Install a C compiler (GCC recommended)
2. Compile any program:

```id="h9k2pl"
gcc program.c -o program
```

3. Run the executable:

```id="v4m8rs"
./program
```
